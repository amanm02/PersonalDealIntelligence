# Ideas Parking Lot

This file stores raw ideas that are not yet ready for roadmap placement or GitHub Issues.

## Rules

- Ideas here are not active scope.
- Keep entries brief.
- Do not assign implementation ownership here.
- Do not use this file as a substitute for GitHub Issues once work is ready.
- Remove or promote stale ideas during roadmap reviews.

## Promotion template

When promoting an idea, create or update a roadmap entry first. Then create a GitHub Issue with:

```markdown
## Objective
## Dependencies
## Product context
## Instructions
## Required behavior
## Acceptance criteria
## Validation
## Non-goals
## Final response requirements
```

The issue body should be directly usable as the implementation prompt. Do not include a separate implementation-prompt section.
